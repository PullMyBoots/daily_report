## 选题背景

本文聚焦于**用户视角的上下文工程**，采用**渐进式梯度设计**，为不同需求的用户提供从轻量级到重量级的完整解决方案。许多用户在长期使用 AI 工具时会遇到"对话越来越长，AI 越来越笨"的问题，但不知道这背后的原理和应对方法。

本文的核心创新在于：**不是一上来就推荐复杂工具，而是从最简单的手动维护 Markdown 文件开始，逐步引导用户根据自己的实际需求升级到更高级的方案。**
