# 测试文章：如何使用文颜MCP自动发布到微信公众号

## 简介

这是一篇测试文章，用于演示如何使用文颜 MCP Server 自动将 Markdown 文章发布到微信公众号草稿箱。

## 功能特点

1. **自动排版**：使用精美的主题自动排版
2. **图片上传**：自动上传本地和网络图片
3. **一键发布**：直接发布到公众号草稿箱

### 代码示例

```python
def hello_world():
    print("Hello, WeChat Official Account!")
    return "发布成功！"
```

## 使用步骤

### 1. 配置环境变量

在 `.env` 文件中配置你的微信公众号 AppID 和 AppSecret：

```env
WECHAT_APP_ID=your_app_id
WECHAT_APP_SECRET=your_app_secret
```

### 2. 准备文章

确保文章包含必要的 frontmatter：

- `title`：文章标题
- `cover`：封面图片（可选）

### 3. 使用 AI 发布

对 AI 助手说："帮我把这篇文章发布到微信公众号"

## 注意事项

> ⚠️ **重要**：必须将服务器 IP 添加到微信公众号平台的 IP 白名单中！

## 总结

文颜 MCP Server 让微信公众号的文章发布变得更加简单和智能化。通过与 AI 助手的配合，可以实现真正的自动化发布流程。
