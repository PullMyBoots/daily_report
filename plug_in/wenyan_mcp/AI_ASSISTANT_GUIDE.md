## 🔄 下一步：重启 Cursor

**重要**：必须重启 Cursor 才能加载新的 MCP 配置。

### 重启方法：

1. **完全退出 Cursor**
   - macOS: `Cmd + Q`
   - 或菜单栏：Cursor → Quit Cursor

2. **重新打开 Cursor**

3. **验证 MCP 已加载**
   - 在 Cursor 中，你应该能看到"公众号助手"工具可用

## 📖 文章格式要求

AI 助手会自动处理大部分格式问题，但你的 Markdown 文章最好包含 frontmatter：

```markdown
---
title: 文章标题
cover: /path/to/cover.jpg  # 可选，如果文章有图片
author: 作者名            # 可选
---

# 文章正文

你的内容...

![示例图片](https://example.com/image.jpg)
```

### 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| `title` | ✅ | 文章标题 |
| `cover` | ❌ | 封面图片（本地路径或URL）<br>如果文章有图片可省略 |
| `author` | ❌ | 作者名称 |
| `digest` | ❌ | 文章摘要（不填会自动生成） |

## 🐛 常见问题

### Q1: AI 说找不到文章文件？

**A**: 使用相对路径或绝对路径：
```
# 相对路径
./example_article.md

# 绝对路径
/Users/dushiliren/Desktop/Self-Media/personal_growth/get_text_vocal/plug_in/wenyan_mcp/example_article.md
```

### Q2: 发布失败怎么办？

**A**: AI 会显示错误信息。常见原因：
- IP 不在白名单（检查微信后台）
- 图片路径不存在（检查图片路径）
- 图片太大（压缩到 < 2MB）

### Q3: 如何知道 MCP 是否正常工作？

**A**: 重启 Cursor 后，对 AI 说：
```
你有没有"公众号助手"这个工具？
```

如果 AI 回答"有"，说明 MCP 配置成功。

### Q4: 可以同时发布多个平台吗？

**A**: 目前 wenyan-mcp 专注于微信公众号。如果需要发布到其他平台，可以：
- 使用 `plug_in/blog_auto_publishing/` 中的其他工具
- 或者让 AI 帮你转换格式后手动发布
